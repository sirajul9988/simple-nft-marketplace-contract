# Simple NFT Marketplace

This repository contains a high-quality, secure smart contract for an NFT Marketplace. It allows users to list their ERC-721 tokens for sale, update prices, or cancel listings. It includes a basic fee mechanism for the marketplace owner.

## Features
- **Listing Mechanism:** Users can list NFTs by approving the marketplace.
- **Atomic Purchases:** Secure transfer of funds and assets in a single transaction.
- **Safe Transfers:** Uses `IERC721` interfaces to ensure compatibility.
- **Flat Structure:** Designed for easy integration and deployment.

## Technical Stack
- Solidity 0.8.20
- OpenZeppelin Contracts
- ERC-721 Standard
